# Проект: Сайт информационных продуктов

Многостраничный статический сайт для продуктов коуча Юлии Перевозниковой:

- Тренинг `«Вырасти Себя Снова»`
- Канал целительных практик
- Индивидуальная трансформационная работа

## Структура проекта

- `RYA_HP/` — основная рабочая версия сайта
- `RYA_HP/index.html` — главная страница с приветственным лендингом, регистрацией и личным кабинетом
- `RYA_HP/training.html` — страница тренинга с разграничением доступа
- `RYA_HP/healing-channel.html` — страница канала практик
- `RYA_HP/therapy.html` — страница индивидуальной работы
- `RYA_HP/css/style.css` — общий файл стилей
- `RYA_HP/js/theme.js` — переключение светлой/тёмной темы
- `RYA_HP/js/registration.js` — регистрация и личный кабинет участника
- `RYA_HP/js/access.js` — уровни доступа и демонстрационная оплата
- `RYA_HP/js/landing.js` — интерактив приветственного лендинга (FAQ и быстрые переходы)

## Как запустить локально

1. Откройте папку проекта в Cursor.
2. Запустите локальный сервер из корня проекта, например:
   - `python -m http.server 5500`
3. Откройте в браузере:
   - `http://localhost:5500/RYA_HP/index.html`

## Функционал

- Переключение темы (светлая/тёмная), сохранение выбора в `localStorage`
- Регистрация участника и валидация формы
- Личный кабинет участника с автоподстановкой данных
- Разграничение доступа: гость / зарегистрирован / оплачено (демо-режим)
- Выбор формата и демонстрационная оплата на продуктовых страницах
- FAQ-аккордеон и быстрые переходы на главной странице

## Примечание

Оплата и хранение данных сейчас реализованы в демонстрационном формате через `localStorage`.
Для production-режима следует подключить backend API, базу данных и платёжный шлюз.
